# AXION Keyboard & Layouts Spec v1

Purpose: make keyboard layout switching and management first-class and fast.

## Features
- Multiple keyboard layouts per user
- One-click layout switch from tray
- Primary layout pin
- Per-app last-used layout memory (optional)
- Shortcut to cycle layouts

## Actions
- Add layout
- Remove layout
- Set primary layout
- Set active layout
- Enable per-app memory

## Done criteria
- Runtime host persists layouts
- Emits corr-traced events
- Integrates with Time & Language category and tray indicator
