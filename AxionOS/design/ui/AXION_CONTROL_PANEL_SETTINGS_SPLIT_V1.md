# AXION Control Panel + Settings Split v1

Purpose: keep classic power-user control while still offering modern Settings UX.

## Principle
- **Settings** = common daily preferences (simple UX)
- **Control Panel** = advanced/admin tools (direct, dense, efficient)

## 1) Axion Settings (consumer path)
- Personalization (theme, wallpaper, taskbar basics)
- Accounts (profile, sign-in options)
- Network quick setup
- Apps & defaults
- Notifications
- Accessibility
- Updates overview

## 2) Axion Control Panel (pro path)
- Devices & Drivers
- Programs & Features
- Services Manager
- Credential & Certificate Manager
- Firewall & Network Policy
- Storage & Disk Tools
- Backup/Restore/Checkpoint Manager
- Event/Trace Viewer
- Local Users & Roles
- Task Scheduler
- Performance Monitor

## 3) UX rule
- Never hide advanced controls behind consumer settings screens.
- Every advanced page has direct launch path + searchable command id.
