# AXION Personalization Category Spec v1

Purpose: close out Personalization with practical, performance-aware controls.

## Sections
- Theme selection (light/dark/custom)
- Accent color + wallpaper linkage
- Desktop icons defaults (profile/main drive/recycle bin)
- Taskbar style (alignment/icon behavior)
- Start menu layout preferences

## Actions
- Apply theme profile
- Set accent mode (manual/auto)
- Set wallpaper pack/profile
- Toggle desktop icon visibility defaults
- Set taskbar alignment

## Done criteria
- Runtime host persists personalization state
- Corr-traced personalization events emitted
- Syncs with settings/taskbar/start/desktop hosts
