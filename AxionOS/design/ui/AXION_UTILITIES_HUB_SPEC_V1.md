# Axion Utilities Hub Spec v1

## Goal
Single launcher for core OS utilities with searchable command palette.

## Name
- Product: **Axion Utilities**
- ID: `axion-utilities`

## v1 utility set
- Pulse Monitor (task/process/resource view)
- Sentinel (security/quarantine view)
- TraceView (audit/event viewer)
- Network Panel
- Storage Inspector
- Service Manager
- Display & Theme
- Axion Capture entry
- Axion Prompt entry

## UX
- Left category rail + right content panel
- Global search at top (`Ctrl+K`)
- "Recent tools" strip

## Non-goal
- No distro-like package tools in v1
