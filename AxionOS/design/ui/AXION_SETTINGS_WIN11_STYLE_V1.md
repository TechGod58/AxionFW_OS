# AXION Settings UX (Windows 11 Style) Spec v1

Goal: keep the modern Windows 11-style settings experience users like, remade cleanly for AxionOS.

## Design language
- Left navigation rail
- Search-first settings
- Card-based category pages
- Breadcrumb + deep-link routes
- Smooth transitions, clear toggles

## Top categories (v1)
- System
- Bluetooth & Devices
- Network & Internet
- Personalization
- Apps
- Accounts
- Time & Language
- Gaming/Media
- Accessibility
- Privacy & Security
- Update & Recovery

## Rules
- Familiar layout behavior
- Axion-native implementation (no proprietary reuse)
- Advanced controls remain available via Axion Control Panel

## v1 done
- major settings categories mapped
- search and deep links functional
- consistency with taskbar/start shell profile
