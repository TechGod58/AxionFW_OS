# AXION Built-in Tools Roadmap v1

Priority tools requested:

1. Snipping Tool (Axion Capture)
2. Utilities Hub (Axion Utilities)
3. Axion Command Prompt (Windows-compatible)

## Product rule
- Original implementation, Windows-familiar UX.
- No proprietary code carryover.
- Full compatibility bridge for Windows workflows where technically/legal safe.

## Delivery order
1) Command Prompt compatibility first (unblocks dev + ops)
2) Snipping Tool (high user-facing value)
3) Utilities Hub (control center wrapper)

4) Axion Notes (OS-level sticky notes)
5) Axion Shell (PowerShell-compatible, Axion-native evolution)
6) Logon Security + Identity Access (OS-level foundation)
