# AXION Enterprise Networking Baseline v1

Purpose: enterprise-ready network setup foundation for managed environments.

## 1) Included in baseline
- VPN protocols/connectors via Access Center model
- Policy-based split/full tunnel
- Device posture checks before connect
- Certificate/key management screen
- Corr-traced session auditing

## 2) Enterprise setup blocks
- Multi-profile VPN policy packs (HQ/Branch/Contractor)
- DNS policy + internal route policy bundles
- Remote desktop profile governance
- Connector health diagnostics and export

## 3) Managed update model
- Signed policy bundle updates
- Staged rollout rings (pilot -> stable -> broad)
- Rollback to previous known-good policy set

## 4) Roadmap extensions
- Domain/IdP federation adapters
- NAC posture integration
- Central policy distribution service

## 5) Current status
- Design coverage exists (Secure Access + Connector Model + Access UI map)
- Implementation still scaffold phase; enterprise runtime services are next build wave
