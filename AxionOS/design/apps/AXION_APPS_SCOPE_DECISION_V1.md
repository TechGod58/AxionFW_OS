# AXION Apps Scope Decision v1

## Email Client
Status: **Deferred (not in v1 core)**

Reason:
- Avoid scope bloat while core OS/security/runtime systems are being established.
- Prioritize built-ins with immediate daily utility and low integration risk.

Revisit trigger:
- After Control Surface + Shell + Access Center + Notes + Calculator ship stable.

## Calculator
Status: **In Scope (v1 core app)**

Rationale:
- High daily utility
- Low runtime risk
- Strong office/home user value when enhanced with finance tabs
