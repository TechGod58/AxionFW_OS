# AXION Accessibility Checklist v1 (App Ship Gate)

## Required before release

- [ ] Full keyboard navigation (no dead ends)
- [ ] Visible focus states on all interactive controls
- [ ] Screen reader labels/roles for all controls
- [ ] Dynamic state announcements (errors/success/progress)
- [ ] Text scaling works without layout breakage
- [ ] High contrast mode pass
- [ ] Reduced motion mode pass
- [ ] Color is not the only information channel
- [ ] Captions/subtitles where media/audio is core
- [ ] Shortcut discoverability/help entry

## Severity policy
- Critical accessibility failures block release.
