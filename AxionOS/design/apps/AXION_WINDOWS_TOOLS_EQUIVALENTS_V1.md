# AXION Windows-Tools Equivalent Matrix v1

Goal: include the practical tools most users/admins actually need in base OS value.

## Must-have in base ($100 value pack)

1. Axion Prompt (CMD-compatible)
2. Axion Shell (PowerShell-compatible)
3. Axion Pad (Notepad equivalent)
4. Pulse Monitor (Task Manager equivalent)
5. Service Manager
6. Device Manager (driver status + rebind)
7. Disk/Storage Inspector
8. Event/Trace Viewer
9. Scheduler (Task Scheduler equivalent)
10. Backup/Restore Manager
11. Registry-like config editor (safe-gated)
12. Certificate/Credential Manager
13. Firewall/Network Policy Manager
14. Calculator (with finance tabs)
15. Clock + Calendar
16. Snipping/Capture
17. Camera (rich settings)
18. PDF View (reader)

## Base if feasible in v1.1
- PDF Studio (editor basics)
- Write (word processor core)
- Sheets (spreadsheet core)

## Add-on upsell candidates (low-cost)
- Advanced PDF editing bundles
- Pro spreadsheet analytics pack
- Creative Studio premium effects packs
- Advanced automation modules

## Rule
Ship handy tools first, candy second.
