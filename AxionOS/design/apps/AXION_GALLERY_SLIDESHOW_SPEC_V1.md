# AXION Gallery + Slideshow Spec v1

## Product
- Name: **Axion Gallery**
- ID: `axion-gallery`

## Goals
- Fast local photo/video browsing
- Beautiful customizable slideshow mode
- Zero-cloud requirement by default

## Core v1 features

1) Library
- folder watch/import
- timeline + album views
- favorites/tags
- duplicate detection (hash-based)

2) Viewer
- zoom/pan/rotate
- metadata panel (EXIF/basic)
- quick edit handoff to Axion Creative Studio

3) Slideshow (customizable)
- transition style (fade, slide, zoom, none)
- transition speed
- per-slide duration
- random/sequential/repeat modes
- soundtrack playlist (optional)
- caption on/off + metadata overlays
- theme presets (light/dark/cinematic)

4) Output
- cast/present mode
- save slideshow preset profiles

## Upsell options (optional, cheap)
- premium transition packs
- cinematic soundtrack/effects packs
- advanced slideshow templates

## Security/privacy
- local-first media index
- no auto-cloud sync
- export/import actions audited

## v1 done
- customizable slideshow fully usable from base product
- stable large-library browsing
- one-click handoff to Creative Studio
