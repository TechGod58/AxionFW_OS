# AXION Media Utilities Pack v1

Purpose: standalone quick tools for common media operations without opening full editor.

## Tools
- Video Splitter
- Video Joiner
- Format Converter
- Version/Codec Profile Changer

## Input safety checks
- file integrity check
- codec/container validation
- size/resource estimate before processing

## Output controls
- destination must use `safe://` mapping
- optional overwrite guard
- checksum generation for exported media

## Presets (starter)
- Mobile compatible
- Web upload optimized
- Archive quality
- Social clip quick export

## v1 done
- all utilities callable from Axion Utilities Hub and context menu
